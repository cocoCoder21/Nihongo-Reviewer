## Grammar Points

### 1. ～べきです / ～べきではない (Should / Should not)

**Meaning:** Ought to ~ / Should ~. Strong moral or logical obligation.

**Formation:** V dictionary + べきです (Exception: する → するべき / すべき)

**Examples:**
- 子供の 個性を 大切にすべきです。 — We should value children's individuality.
- いじめを 見たら、先生に 報告するべきです。 — If you see bullying, you should report it.
- テストの 結果だけで 判断すべきではありません。 — We shouldn't judge only by test results.

---

### 2. ～に関して / ～に関する (Regarding ~ / Concerning ~)

**Meaning:** Regarding ~. More formal than ～について.

**Formation:** N + に関して / に関する + N

**Examples:**
- 教育制度に関して いろいろな 意見が あります。 — There are various opinions regarding the education system.
- 入試に関する 情報を 集めています。 — I'm gathering information regarding entrance exams.

---

### 3. ～を中心に (Centered on ~ / Focusing on ~)

**Meaning:** With ~ as the center/focus.

**Formation:** N + を中心に / を中心として

**Examples:**
- 日本の 教育は 受験を中心に 行われてきました。 — Japanese education has been conducted centering on entrance exams.
- 東京を中心に 人口が 集中しています。 — Population is concentrated centering on Tokyo.

---

### 4. ～どころか (Far from ~ / Let alone ~)

**Meaning:** Far from ~ / On the contrary. Stronger denial + unexpected reality.

**Formation:** N / V plain + どころか

**Examples:**
- 勉強どころか、遊ぶ 時間も ありません。 — Far from studying, I don't even have time to play.
- 成績が 上がるどころか、下がってしまいました。 — Far from improving, my grades actually went down.

## Mini Quiz

**Q1.** Say "Students should study not just for exams" using ～べきです.

**Q2.** Express "I'd like to hear opinions regarding this policy" using ～に関して.

**Q3.** Say "Education is conducted focusing on English" using ～を中心に.

**Q4.** Express "Far from understanding, I can't even read the textbook" using ～どころか.

**Q5.** Compare: "Japanese education has advantages and disadvantages compared to other countries."
