## Grammar Points

### 1. ～ば Conditional (仮定形)

Expresses a general or logical condition: "If ~, then ~."

**Formation:**

| Type | Rule | Example |
|------|------|---------|
| I (う-verbs) | Change -u → -eba | 行く → 行けば、読む → 読めば、話す → 話せば |
| II (る-verbs) | Drop -る, add -れば | 食べる → 食べれば、見る → 見れば |
| III | する → すれば、来る → 来れば（くれば） | — |
| い-adj | Drop -い, add -ければ | 高い → 高ければ、安い → 安ければ |
| な-adj | + であれば / なら（ば） | 静か → 静かであれば / 静かなら |
| Noun | + であれば / なら（ば） | 日曜日 → 日曜日であれば / 日曜日なら |
| Negative | -ない → -なければ | 行かない → 行かなければ |

**Examples:**
- 春に なれば、桜が 咲きます。 — If spring comes, the cherry blossoms bloom.
- この ボタンを 押せば、お釣りが 出ます。 — If you push this button, change comes out.
- 安ければ、買います。 — If it's cheap, I'll buy it.
- もし 時間が あれば、来て ください。 — If you have time, please come.

---

### 2. ～なければ なりません (Must ~)

Literally "if you don't ~, it won't do." Means **must / have to**.

- 毎日 薬を 飲まなければ なりません。 — I must take medicine every day.
- 9時までに 来なければ なりません。 — I must come by 9 o'clock.

---

### 3. ～ば vs Other Conditionals (Summary)

| Form | Nuance |
|------|--------|
| ～ば | General/logical condition, hypothetical |
| ～たら | Personal, one-time events, "when" |
| ～と | Automatic/natural result, habit |
| ～なら | Contextual, "if it's the case that" |
