| 集まる (あつまる) | 集める (あつめる) | gather / collect |
| 見つかる (みつかる) | 見つける (みつける) | be found / find |
| 届く (とどく) | 届ける (とどける) | arrive / deliver |
| 焼ける (やける) | 焼く (やく) | burn / bake |
| 沸く (わく) | 沸かす (わかす) | boil |

> 他動詞 + てあります = intentional state (窓が**開けて**あります → The window has been opened on purpose)

## 付録 E — そうです の使い分け Distinguishing ～そうです

| Type | Meaning | Formation | Example | Lesson |
|------|---------|-----------|---------|--------|
| **Appearance** | Looks like ~ | V stem + そう / Adj (drop い/な) + そう | おいしそうです | 43 |
| **Hearsay** | I heard that ~ | Plain form + そうです | おいしいそうです | 47 |

> おいし**そう**です = It **looks** delicious (I'm seeing it)
> おいしい**そう**です = I **heard** it's delicious (someone told me)

## 付録 F — 条件表現の比較 Conditional Expressions Compared

| Form | Nuance | Example | Lesson |
|------|--------|---------|--------|
| **～と** | Natural/automatic result, habit | ボタンを 押すと、ドアが 開きます | 23 |
| **～たら** | If/when (one-time, completed) | 雨が 降ったら、行きません | 25 |
| **～ば** | General/logical condition | 安ければ、買います | 35 |
| **～なら** | "If that's the case" (topic-based) | 京都なら、秋が いいですよ | — |
| **～ても** | Even if (concessive) | 雨が 降っても、行きます | 25 |

### When to Use Which

| Situation | Best Form |
|-----------|-----------|
| Scientific fact / machine operation | ～と |
| Specific future plan | ～たら |
| General advice / logical condition | ～ば |
| Responding to someone's topic | ～なら |
| Expressing "regardless" | ～ても |
